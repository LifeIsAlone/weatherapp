# 🌦 Open API를 이용해 만든 날씨 웹사이트

## 🚩Page

https://lifeisalone.github.io/weatherapp/

## Open API

- Unsplash API
- OpenWeather API

## 서비스 소개
- 실시간 날씨 데이터 및 7일간의 날씨 데이터 제공
- 도시 검색으로 해당 도시의 날씨 데이터 및 사진 제공
- 섭씨, 화씨 단위 변경 및 다크모드 설정
- 자외선 세기 및 일출, 일몰 시간 등 추가 날씨 데이터 확인
